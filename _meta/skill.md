# 🛠️ Skill Tracker

> **Why this file exists:**
> When you're learning for 6 months, it's easy to feel like you're not improving and forget what you've built.
> This file tracks every skill you've gained, at what level, and proves it with a real project link.
> In a job interview, you can look here and confidently say "I built X with Y — here's the link."
> It also helps your AI know exactly what you can and can't do, so it doesn't over-explain things you know or skip things you don't.

---

## 📋 How to Use This File

1. When you **START** learning a skill → mark it: 🔵 Aware
2. When you've **touched it** but still look things up → mark it: 🟡 Learning
3. When you've **used it in a project** → mark it: 🟠 Comfortable
4. When you can **explain it to someone else** without help → mark it: 🟢 Confident
5. Always add a **Proof link** (GitHub repo or file) and **Last Practiced** date

**Update this file:** Every Thursday (per your weekly routine) and at the end of each project.
